Overview

Whole-Frame Tiled Print is a macOS application for printing large images—calligraphy, paintings, photography, and other artwork—by splitting them across multiple standard paper sheets. You can print at true physical size or scale the image to fit a chosen number of sheets with intelligent layout and margins.

All processing happens on your Mac. Your images are never uploaded to our servers.

Quick Start

  1. Click Open Image or drag an image file (PNG, JPEG, TIFF, HEIC) into the window.
  2. Choose a Print Mode (see below) and set paper size, orientation, and units.
  3. Review the tiled preview and status bar. Adjust overlap or margins if needed.
  4. Export sheets as separate files (filename includes row/column, e.g. Artwork_R1C1.pdf) or a combined PDF, or click Print to send to your system printer.
  5. Trim and assemble printed sheets using the overlap zones and alignment marks on the physical printout.

Print Modes

Actual Size (100%)

Prints the image at a specified physical width. The app calculates how many sheets are required and tiles the image at true scale. Set Artwork Width, Tile overlap (for gluing alignment), and Outer assembly margin when centering is enabled.

Fit to Sheets · Smart Scale

Scales the entire image to fit a chosen grid of sheets (e.g. 2×3). Use Auto-fit to aspect ratio to suggest a grid. Golden-ratio margins can be applied for a balanced border around the artwork.

Assembly Options

  • Center image in assembly — Centers the artwork in the sheet grid. Only outer-edge sheets keep margins; inner sheets print edge-to-edge for easier assembly without trimming interior seams.
  • Outer assembly margin — Safe border on the perimeter of the assembled piece.
  • Preview marks — Crop marks and page labels (R·C) appear in the on-screen preview only. Printed and exported files contain clean output without guides or on-page numbering. Sheet position is encoded in export filenames (_R1C1, _R2C3, etc.).

Supported Paper

A5, A4, A3, A2, A1, Letter, Legal, and Tabloid — portrait or landscape. Units: millimeters, centimeters, or inches.

FAQ

Why is effective resolution shown in orange?

Values below 150 dpi may look soft when printed large. Use a higher-resolution source image or reduce the output size.

Can I use network printers?

Yes. The app uses macOS printing services. Add printers in System Settings → Printers & Scanners, then refresh the list in the app.

Does the app require the internet?

No. Core features work offline. Network access is used only for communicating with network printers through the system print subsystem, not for sending your images to us.

概述

整幅拼接打印（Whole-Frame Tiled Print） 是一款 macOS 应用，可将书法、国画、摄影等大幅作品分割到多张标准打印纸上进行拼接输出。支持按真实物理尺寸打印，或在指定数量的纸张上智能缩放布局。

所有处理均在您的 Mac 本地完成，图片不会上传至我们的服务器。

主界面

主界面：左侧设置栏、中央拼版预览、底部状态栏、右上方工具栏。

① 侧栏
加载作品、选择模式、纸张、拼接与打印机。
② 预览
实时显示每张纸的拼版效果。
③ 状态栏
张数、总尺寸、缩放、分辨率、留白与重叠。
④ 工具栏
打开、导出、打印。

快速开始

  1. 点击打开图片，或将 PNG/JPEG/TIFF/HEIC 文件拖入窗口。
  2. 选择打印模式，设置纸张大小、方向与单位。
  3. 查看预览与状态栏，按需调整重叠与留白。
  4. 导出分张文件（文件名含行列序号，如 作品_R1C1.pdf）或合并 PDF，或打印至系统打印机。
  5. 按重叠区域对齐，裁切并拼合各张纸。

打印模式

原尺寸拼接 (100%)

按指定物理宽度 1:1 打印，自动计算所需纸张数量。可设置作品宽度、拼接重叠与外圈留白。

指定纸张 · 智能缩放

在选定的行列网格上缩放整幅画面，可按比例智能匹配张数，并支持黄金分割留白。

拼接选项

  • 画面居中拼接 — 仅最外圈纸张保留留白，内部纸张满版打印，便于拼合。
  • 预览标记 — 裁切线与页码标签仅显示在屏幕预览中；打印与导出的文件为干净输出，序号体现在文件名中（_R1C1 等）。

常见问题

有效分辨率显示橙色？

低于 150 dpi 时大幅打印可能偏软，请使用更高分辨率原图或缩小输出尺寸。

需要联网吗？

核心功能可离线使用。网络权限仅用于系统与网络打印机通信，不会上传您的图片。

Aperçu

Whole-Frame Tiled Print est une application macOS pour imprimer de grandes images — calligraphie, peinture, photographie — en les divisant sur plusieurs feuilles standard. Impression à taille réelle ou mise à l'échelle intelligente sur une grille choisie.

Tout le traitement se fait localement sur votre Mac. Vos images ne sont jamais téléversées sur nos serveurs.

Fenêtre principale

Barre latérale (gauche), aperçu (centre), barre d'état (bas), barre d'outils (haut droite).

Démarrage rapide

  1. Ouvrez une image (PNG, JPEG, TIFF, HEIC) ou glissez-déposez-la.
  2. Choisissez le mode, le format de papier et l'orientation.
  3. Vérifiez l'aperçu et exportez ou imprimez.
  4. Les fichiers exportés incluent la position dans le nom (_R1C1).

Export et impression

L'impression et l'export n'incluent pas de lignes pointillées, marques de coupe ni numéros sur la page. Les repères n'apparaissent que dans l'aperçu à l'écran.
